Maryknoll Taiwanese English Dictionary Search

This program accepts Taiwanese word and opens appropriate page of 'Maryknoll Taiwanese English Dictionary' which is included in this program in PDF format.

Use '^' for nasal monothong 'ⁿ', and use 'ou' for 'o͘'.

If you enter...

  • '0' : it will open 'Foreword' section of the dictionary
  • '1' : it will open 'Major Noun-Classifiers Used in Spoken Taiwanese'
  • '2' : it will open 'Names of Christian Missions'
  • '3' : it will open 'Kinship Terms'
